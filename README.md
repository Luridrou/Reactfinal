# Para poder usar el proyecto

### Seguir los siguientes pasos:

- Clonar el repositorio de GitHub:

  ```bash
  git clone 
  ```

- cd dentro del directorio de la carpeta:
  ```bash
  cd Proyecto_Ecommerce_React
  ```

- Instala las dependencias.
  ```bash
  npm install
  ```

- Para correr el proyecto:

  ```bash
  npm run start

  npm start
  ```
